# 贪吃蛇游戏 - Claude Demo

一个基于HTML5和JavaScript的经典贪吃蛇游戏实现。

## 🎮 游戏特性

- **经典玩法**: 传统的贪吃蛇游戏机制
- **响应式设计**: 支持不同屏幕尺寸
- **键盘控制**: 使用方向键控制蛇的移动
- **计分系统**: 实时显示当前得分
- **游戏结束检测**: 撞墙或撞到自己时游戏结束

## 📁 文件结构

```
.
├── snake_game.html        # 基础版本

```

## 🚀 快速开始

1. **克隆项目**:
   ```bash
   git clone https://github.com/your-username/claude-demo.git
   ```

2. **运行游戏**:
   - 直接双击 `snake_game_final.html` 文件
   - 或在浏览器中打开该文件

3. **控制方式**:
   - ↑ 上方向键: 向上移动
   - ↓ 下方向键: 向下移动
   - ← 左方向键: 向左移动
   - → 右方向键: 向右移动
